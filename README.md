# О чём репозиторий

Это тестовое cli приложение для обработки csv файлов с метриками YouTube видео и формированием отчётов по заданным параметрам.

# Что используется

Из сторонних библиотек в проекте используется pytest, tabulate, ruff, *setuptools. 

> setuptools устанавливается для сборки с pyproject.toml. Изначально я подумал отказаться от установки проекта как пакета из-за требования использовать только стандартные библиотеки, однако в FAQ указано, что можно *использовать что-то кроме pip для управления зависимостями*. Считаю, что данный кейс подходит, и проект так выглядит приятнее.

# Общая логика

`cli.py` - точка входа. `app.py` - оркестрация функционала.


# Примеры запуска

Команда `report-cli`:

```
report-cli --files tests/data/stats1.csv --report clickbait
```
