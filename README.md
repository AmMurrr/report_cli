# О чём репозиторий

Это тестовое cli приложение для обработки csv файлов с метриками YouTube видео и формированием отчётов по заданным параметрам.

# Что используется

Из сторонних библиотек в проекте используется pytest, tabulate, ruff. 

# Установка

Установка с тестами и линтером
```
pip install -e ".[dev]"
```

Установка только с необходимым для вывода `tabulate`
```
pip install -e .
```
